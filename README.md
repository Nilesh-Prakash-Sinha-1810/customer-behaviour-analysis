# Customer Behavior Analysis

## 📊 Objective
Analyze customer purchase data to understand spending patterns, popular products, and business insights.

## 🛠 Tools Used
- Python
- Pandas

## 📁 Dataset
The dataset contains:
- Customer ID
- Product
- Category
- Purchase Amount
- Date

## 🔍 Key Analysis
- Total spending per customer
- Top 2 customers
- Most popular product
- Revenue by category
- Daily sales trends
- Repeat customers
- Average order value

## 📈 Key Insights
- Customers 2 and 3 are the highest spenders
- Product A is the most popular
- Home category generates the highest revenue
- Sales peak on 2024-01-10
- Several customers make repeat purchases

## 🧠 Conclusion
The business should focus on high-value customers and high-revenue categories to maximize growth.
